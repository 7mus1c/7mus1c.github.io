# AI

AI 是让机器模仿人类智能行为（学习、推理、感知、决策）的科学与工程。它不是通过硬编码的规则，而是通过从数据中学习模式来完成任务。AI 的目标是让机器能够自主地执行任务，而不需要人类干预。

## 核心名词

### 机器学习 (Machine Learning) ML

机器学习是一种让计算机通过模型分析大量数据来学习，并且给予学习的结果做出预测或者决策。

### 深度学习 (Deep Learning) DL

深度学习，模仿人脑的神经结构，使用包含多个层次的神经网络来处理数据。能够从原始数据（图像像素、音频波形）中提取非常复杂的特征，无需过多人工干预。比如：图像识别、语音识别、自然语言处理等。

### 神经网络 (Neural Network) NN

神经网络是深度学习的基础架构。由像神经单元一样的节点层层连接而成。数据从输入层流入，经过多哥隐藏层的处理最后输出层得到结果。

### 生成式 AI (Generative AI)

生成式 AI 是一种 AI 技术，它能够生成新的、与训练数据相似的数据。比如：生成新的图像、生成新的文本等。

### 计算机视觉 (Computer Vision) CV

让计算机能看和理解图像和视频，比如：人脸识别、图像分类、自动驾驶中的物体检测等。

### 自然语言处理 (Natural Language Processing) NLP

让计算机能理解和解释、生成人类语言，比如：机器翻译、语音识别、情感分析、智能客服、语音助手。

### API 接口

在 AI 环境下，通常是指云服务商提供的 AI 能力接口，自己不需要训练模型，只需要调用 API，发送数据就能得到 AI 处理后的结果。比如：调用 openAI 的 API，发送一段文字，就能得到 AI 生成的文字。

### 大语言模型 (Large Language Model) LLM

大语言模型是一种基于深度学习的自然语言处理模型，它能够理解和生成人类语言。比如：GPT-3、BERT、T5 等。

### 提示词工程 (Prompt Engineering)

设计和优化输入给 AI 模型（尤其是 LLM）的提示词，以获得更好的输出结果。比如在百度输入“今天天气怎么样”，百度会返回天气信息，而输入“今天天气怎么样，温度多少度”，百度会返回更详细的信息。

### Token

在 NLP（自然语言处理）中，Token 是指将文本分割成最小单位，比如将一句话分割成单词或者字符。英文可能是一个单词或者词根，中文通常是一个字或者一个词。

### LangChain

一个用户开发由 LLM 驱动的应用程序的框架。它能轻松的将 LLM 与外部数据源、API、数据库、工具等连接起来，构建复杂的 AI 应用。比如：将 LLM 与数据库连接起来，实现一个智能客服系统。又或者将 LLM 与搜索引擎连接起来，实现一个智能问答系统。

### RAG （Retrieval-Augmented Generation）检索增强生成

一种结合了外部知识库（比如数据库、文档）与 LLM 的生成能力结合的技术。通过检索知识库中的相关内容，再结合 LLM 的生成能力，实现更准确和丰富的输出结果。
它的出现是为了解决 LLM 的胡编乱造和知识过时或者不全面的问题。

## 与前端结合

对于前端开发来说，并不需要成为训练神经网络的专家，但是要成为 **AI 能与用户界面之间的桥梁**。
任务是理解 AI 能做什么，如何使用 AI，如何将 AI 与用户界面结合。
